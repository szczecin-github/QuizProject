# Java Spring Boot Quiz System

A console-based Quiz application using Java 17, Spring Boot, and H2 Database.

## Features
- **Spring Boot & JPA:** Automates database handling and object mapping.
- **H2 Database:** In-memory database for storing questions (reset on restart).
- **Maven:** Manages dependencies and build process.

## Prerequisites
- Java JDK 17 or higher
- Maven (optional, if using the wrapper)
