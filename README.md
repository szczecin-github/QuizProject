# Java Quiz System

A console-based Quiz application
## How to Run

1. **Prerequisites:**
   - Java Development Kit (JDK) installed.
   - [Gson 2.10.1](https://repo1.maven.org/maven2/com/google/code/gson/gson/2.10.1/gson-2.10.1.jar) library.

2. **Setup:**
   - Download the `gson-2.10.1.jar` file.
   - Place it in the root folder of this project.

3. **Compile:**
   - **Windows:** `javac -cp ".;gson-2.10.1.jar" *.java`
   - **Mac/Linux:** `javac -cp ".:gson-2.10.1.jar" *.java`

4. **Run:**
   - **Windows:** `java -cp ".;gson-2.10.1.jar" Main`
   - **Mac/Linux:** `java -cp ".:gson-2.10.1.jar" Main`
